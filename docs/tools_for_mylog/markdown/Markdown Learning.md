#### 2.2.1 Arithmatex
!!! note
    Arithmatex 扩展允许渲染块和内联块方程，并与数学排版库 MathJax1 无缝集成。

- 使用块语法

当输入为：
`$$
\cos x=\sum_{k=0}^{\infty}\frac{(-1)^k}{(2k)!}x^{2k}
$$`

渲染过后的效果：

$$
\cos x=\sum_{k=0}^{\infty}\frac{(-1)^k}{(2k)!}x^{2k}
$$

- 使用内联公式

将需要用公式显示的语句使用 `$...$` 或者 `\(...\)`囊括起来
